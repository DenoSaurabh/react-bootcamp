# Goal of Project

The Goal was to learn complete React Ecosystem.
Everything I learn in this course is here:

* ReactJS
* Redux
* Routing
* Styled Components
* Firebase
* Authentication & Database
* Persistence
* State Normalization
* Patterns
* Redux Saga
* React Hooks
* Context API
* Apollo + GraphQL
* Webpack + Babel

* Performance
* PWA
* Security
